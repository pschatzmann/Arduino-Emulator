Detailed Description

Arduino-compatible Servo library for Raspberry Pi using pigpio.

Controls RC servo motors via GPIO PWM pulses (50 Hz, 500–2500 µs pulse width). Follows the standard Arduino Servo API.

Usage:

Servo myServo;
myServo.attach(18); // GPIO pin 18
myServo.write(90); // Move to 90 degrees
